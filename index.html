git config - Урозділі початку роботи вказує на ім'я та пошту

git init- Створює новий репозиторій

git status- Відображає стан робочого каталогу

git add- Використовуються для запису версій проектів в історію репозиторію

git commit- Повернути з редагування індексовані зміни до репозиторію

git log- Форматування виводу команди журналу

git checkout- Щоб об'єднати дві гілки, над якими ви працювали

git branch- Дозволяє вам створювати, перераховувати, перейменовувати та видаляти гілки

git push- Використовується для завантаження вмісту локального репозиторію у віддалене сховище

git pull- Використовується для витягування та завантаження вмісту з віддаленого сховища та негайного оновлення

git merge- Об'єднує всі інтегровані зміни в один коміт, а не зберігає їх як окремі коміти

git clone- Копіює існуючий git-репозиторій

git ignore- Файл, який Git було явно наказано ігнорувати

git revert

git reset

git rebase

git cherry-pick

git stash
